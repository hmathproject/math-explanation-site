# 解の配置（2解が同符号）— 判別式・関数値・軸の3条件で解の位置を確定する

---

## 問題

次の問いに答えよ。

方程式 \\( x^2 + (a-3)x + a = 0 \\) の2解がともに正となる実数 \\( a \\) の範囲を求めよ。また、2解がともに負となる実数 \\( a \\) の範囲も求めよ。

---

## 解の配置とは何を求めているか

この問題では、方程式を解いて \\( x \\) の値を求めるのではなく、「2解がともに正（または負）」という条件から、パラメータ \\( a \\) の範囲を逆算します。

\\( f(x) = x^2 + (a-3)x + a \\) と置くと、方程式の解は放物線 \\( y = f(x) \\) とx軸との交点の \\( x \\) 座標です。「2解がともに正」とは、その交点が2つとも \\( x > 0 \\) の範囲にあることを意味します。

---

## 2解がともに正になる3条件

放物線 \\( y = f(x) \\) が「\\( x > 0 \\) の範囲に2点でx軸と交わる」ためには、次の3条件がすべて必要です。

まず下の図で、「ともに正」（左）と「ともに負」（右）の放物線の形がどう違うかを確認してください。

![解の配置（2解が同符号）：ともに正とともに負の放物線配置](/assets/images/quadratic-root-location-same-sign-combined.png)

**条件1: 判別式 \\( D \\geq 0 \\)**

実数解が2つ（または重解）存在するための条件です。\\( D < 0 \\) では放物線がx軸と交わらず、解が実数になりません。

**条件2: \\( f(0) > 0 \\)**

\\( f(0) > 0 \\) は「\\( x = 0 \\) での放物線の値が正」すなわち「放物線が \\( x = 0 \\) でx軸より上にある」ことを意味します。これが成り立てば、\\( x = 0 \\) は2解の外側（2解はともに正の側）にあることが保証されます。

**条件3: 軸 \\( > 0 \\)**

軸（頂点の \\( x \\) 座標）は2解の中点です。軸が正の範囲にあれば、2解の平均が正、すなわち2解はともに正の方向にある可能性が確保されます。

---

## なぜ3条件すべてが必要か

条件1だけでは「実数解が存在する」ことしかわかりません。条件2だけでは「\\( x = 0 \\) が2解の外にある」ことしかわかりません。条件3だけでは「解の中点が正の側にある」ことしかわかりません。

3条件のうち1つでも欠けると、条件を満たさない配置の放物線が混入します。たとえば条件2を外すと「2解ともに負」の放物線も条件1・3を満たすケースがあります。

---

## 2解がともに負になる条件

2解がともに負のときは、条件2と条件3の向きが変わります。

- 判別式 \\( D \\geq 0 \\)（変わらず）
- \\( f(0) > 0 \\)（\\( x = 0 \\) が2解の外側 → 条件2は同じ）
- 軸 \\( < 0 \\)（2解の中点が負の範囲）

この対称性から、「ともに正」と「ともに負」の条件は軸の不等号だけが逆になる構造だとわかります。

---

## まとめ

<div>
$$
\text{2解がともに正} \iff D \geq 0 \quad \text{かつ} \quad f(0) > 0 \quad \text{かつ} \quad \text{（軸）} > 0
$$
</div>

<div>
$$
\text{2解がともに負} \iff D \geq 0 \quad \text{かつ} \quad f(0) > 0 \quad \text{かつ} \quad \text{（軸）} < 0
$$
</div>

各条件は「実数解の存在」「解が0の外側にある」「解が正・負どちらの側にあるか」をそれぞれ独立に保証しています。
