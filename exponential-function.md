# 指数関数のグラフと方程式・不等式

**底の値で増減が変わる理由をグラフで見る**

## この単元で学ぶこと

指数関数 \\( y = a^x \\)（\\( a \\) は底、\\( a > 0 \\) かつ \\( a \neq 1 \\)）のグラフは、底 \\( a \\) の値によって増加するか減少するかが変わります。

- \\( a > 1 \\) のとき：\\( x \\) が大きくなるほど \\( y \\) も大きくなる（右上がり）
- \\( 0 < a < 1 \\) のとき：\\( x \\) が大きくなるほど \\( y \\) は小さくなる（右下がり）

この単元では、

- **なぜ底の値でグラフの形が変わるのか**
- **指数方程式・不等式で「底をそろえる」と何が見えるのか**
- **不等号の向きが逆になる理由は何か**

を、グラフと言葉で順番に確認しながら学びます。

---

## この解説の特徴

「底をそろえて解く」手順を示すだけの解説は多くあります。

この解説では、手順を追うことと同時に、**なぜ底の値で増減が決まるのか**・**なぜ不等号の向きが変わるのか**を、グラフを見ながら確認します。

指数関数のグラフが単調増加か単調減少かという性質は、指数不等式を解くときの「不等号の向きの決まり方」と直結しています。式の意味をグラフと結びつけて理解することを、この解説では大切にしています。

---

## 解説記事

### [指数関数のグラフ — 底の値で増減がどう変わるか](/exp-function-graph/)

底 \\( a \\) が 1 より大きいか小さいかで、グラフの形がどう変わるかを確認します。

\\( y = 2^x \\) と \\( y = \left(\frac{1}{2}\right)^x \\) を例に、2つのケースを横並びで比較します。漸近線 \\( y = 0 \\) の意味と、どの底でも \\( x = 0 \\) で \\( y = 1 \\) になる理由を確認します。

---

### [指数関数の平行移動と対称移動](/exp-function-transform/)

\\( y = a^{x-p}+q \\) の形（平行移動）と y 軸対称 \\( y = a^{-x} = \left(\frac{1}{a}\right)^x \\) の関係を確認します。x 軸方向・y 軸方向への移動が式のどの部分に現れるか、また底の逆数をとると y 軸対称になることを、図で読み取ります。

---

### [指数方程式・不等式 — 底をそろえて解く](/exp-function-equation/)

底をそろえると指数が直接比較できること、\\( a > 1 \\) と \\( 0 < a < 1 \\) で不等号の向きが変わる理由をグラフの単調性から確認します。置換 \\( t = a^x \\)（\\( t > 0 \\)）を使う方程式の解き方も扱います。
