# 軌跡の求め方 — 式変形と適用条件の確認

---

## 媒介変数の消去

動点 \\( P(x, y) \\) の位置がパラメータ \\( t \\) を使って

<div>
$$
x = f(t), \quad y = g(t)
$$
</div>

と表されているとき、\\( t \\) を消去して \\( x \\) と \\( y \\) だけの関係式を求めます。

消去の方針:
- \\( t \\) について一方の式を解いて他方に代入する
- 両式から \\( \sin t, \cos t \\) の形を作り \\( \sin^2 t + \cos^2 t = 1 \\) を利用する

消去した後、パラメータの範囲（例えば \\( t > 0 \\) など）から \\( x, y \\) の取りうる範囲が制限されることがあります。

---

## なぜ「逆の確認」が必要か

**必要条件と十分条件の違い**が核心です。

- **必要条件**: 「動点 P の条件を満たす → 求めた式を満たす」（式変形は必要条件を示す）
- **十分条件**: 「求めた式を満たす → 動点 P の条件を満たす」（逆の確認で検証）

両方が成り立って初めて「軌跡はこの式である」と言えます。式変形だけでは十分条件が保証されていません。

**具体例**: 分母を払う操作では、払った分母がゼロの点が新たに「式を満たす点」として混入する可能性があります。このような点は元の条件を満たさないので除外点として取り除く必要があります。

---

## 典型的な除外点の発生状況

| 状況 | 除外点が生まれる理由 |
|---|---|
| 分母が 0 の点 | 分母を払う前の式が定義されなかった点 |
| 絶対値を外すとき | 符号の条件が失われ、反対符号の点が混入 |
| 変数の範囲制限 | パラメータの範囲制限から \\( x, y \\) の範囲が制限される |
| 因数分解して約分 | 約分した因数が 0 の点（元の式では0/0になる） |

## まとめ

軌跡を求める手順を整理します。

1. 動点 \\( P(x,y) \\) の条件を式に翻訳する
2. パラメータを消去して \\( x, y \\) の関係式を作る（必要条件）
3. 逆を確認する（十分条件の検証）
4. 除外点を取り除いて軌跡を確定する

「式を求めて終わり」ではなく、**逆の確認と除外点の確認**まで行うことが軌跡の問題の完答条件です。
