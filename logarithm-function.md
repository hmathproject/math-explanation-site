# 対数関数のグラフと方程式・不等式

指数方程式 \\( a^x = M \\) の解に「対数」という名前をつけると \\( x = \log_a M \\) と書けます。この対応を逆関数として捉えると、指数関数 \\( y = a^x \\) と対数関数 \\( y = \log_a x \\) はグラフ上で \\( y = x \\) について対称な関係にあります。この単元では、対数の定義からグラフの形・性質・方程式不等式まで順番に確認します。
