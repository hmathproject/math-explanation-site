# 二次関数

二次関数 \\( y = ax^2 + bx + c \\) のグラフは放物線です。軸の位置・頂点の座標・x 軸との交わり方など、グラフの形から読み取れる情報が、最大値・最小値・不等式・解の条件といった問題に直結しています。この分野では、式を追うだけでなく「なぜその形になるのか」をグラフと言葉で順番に確認します。
