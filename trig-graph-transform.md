# グラフの振幅・周期・平行移動

---

## 問題

次のグラフの振幅・周期・位相のずれを求めよ。

(1) \\( y = 3\sin 2\theta \\)　　(2) \\( y = -2\cos\!\left(\theta - \frac{\pi}{3}\right) + 1 \\)

---

## まず何を見るか

\\( y = A\sin(B\theta + C) + D \\) の形では、A・B・C・D の4つのパラメータがそれぞれ独立した変換を担っています。図でその効果を確認します。

次の図で、左から「振幅の変化（A）」「周期の変化（B）」「位相のずれ（C）」「垂直移動（D）」の4つを確認してください。

![4つのパネル：振幅・周期・位相のずれ・垂直移動の比較図](/assets/images/trig-graph-transform-combined.png)

灰色の破線が基準の \\( y = \sin\theta \\)、濃青が変換後のグラフです。各パラメータが独立してグラフの「どの部分」を変えるかが一目で分かります。

---

## なぜ A, B, C, D がそれぞれ独立に働くか

\\( y = A\sin(B\theta + C) + D \\) は \\( y = \sin\theta \\) に対して順番に変換を重ねたものです。

1. \\( \sin\theta \to \sin(B\theta) \\)：x 軸方向に \\( \frac{1}{B} \\) 倍（周期が \\( \frac{1}{B} \\) 倍になる）
2. \\( \sin(B\theta) \to \sin(B\theta + C) \\)：x 軸方向に \\( -\frac{C}{B} \\) シフト（位相のずれ）
3. \\( \sin(B\theta+C) \to A\sin(B\theta+C) \\)：y 軸方向に A 倍（振幅が \\( |A| \\) になる）
4. \\( \to A\sin(B\theta+C) + D \\)：y 軸方向に D シフト（中心線が \\( y = D \\) になる）

---

## 場合別の計算

**(1) \\( y = 3\sin 2\theta \\)**

\\( A = 3,\ B = 2,\ C = 0,\ D = 0 \\)

- 振幅：\\( |A| = 3 \\)（y の範囲は \\( [-3, 3] \\)）
- 周期：\\( \frac{2\pi}{|B|} = \frac{2\pi}{2} = \pi \\)
- 位相のずれ：なし
- 垂直移動：なし

\\( y = \sin\theta \\) の波が \\( [0, \pi] \\) の中に1回分入り、高さが3倍になった形。

**(2) \\( y = -2\cos\!\left(\theta - \frac{\pi}{3}\right) + 1 \\)**

\\( A = -2,\ B = 1,\ C = -\frac{\pi}{3},\ D = 1 \\)

- 振幅：\\( |A| = 2 \\)
- 周期：\\( \frac{2\pi}{1} = 2\pi \\)
- 位相のずれ：\\( \theta = \frac{\pi}{3} \\) だけ右にずれる（cos の最大点が \\( \theta = \frac{\pi}{3} \\) に）
- 垂直移動：中心線が \\( y = 1 \\)
- \\( A < 0 \\)：上下反転

<div>
$$
\text{最大値} = 1 + 2 = 3 \quad\left(\theta = \frac{\pi}{3} + \pi = \frac{4\pi}{3} \text{ のとき}\right)
$$
$$
\text{最小値} = 1 - 2 = -1 \quad\left(\theta = \frac{\pi}{3} \text{ のとき}\right)
$$
</div>

\\( A = -2 \\) なので cos が最大値を取るところ（位相のずれ後の \\( \theta = \frac{\pi}{3} \\)）で \\( y \\) は最小になります。

---

## まとめ

| パラメータ | グラフへの効果 |
|---|---|
| \\( A \\) | 振幅 \\( = \vert A\vert \\)。\\( A < 0 \\) なら上下反転 |
| \\( B \\) | 周期 \\( = \frac{2\pi}{\vert B\vert} \\)。\\( \vert B\vert > 1 \\) で周期が短くなる |
| \\( C \\) | 位相のずれ \\( = -C/B \\)（x 軸方向のシフト） |
| \\( D \\) | 垂直移動（中心線が \\( y = D \\) になる） |

4つのパラメータを分離して読めば、複雑に見える式でも「どのグラフが何倍・何ずれているか」が見えます。
