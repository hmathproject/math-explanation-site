# 増減表と極値の求め方

---

## 問題

次の関数の極値を求め、増減を調べよ。

(1) \\( f(x) = -x^3 + 3x \\)　　(2) \\( f(x) = x^3 \\)

---

## まず何を見るか

\\( f'(x) \\) の符号が \\( + \\) から \\( - \\) に変わる点では**極大**、\\( - \\) から \\( + \\) に変わる点では**極小**が成立します。ただし \\( f'(c) = 0 \\) であっても符号が変わらなければ極値にはなりません。

次の図で、符号変化ありの場合（左）と符号変化なしの反例（右）を先に確認してください。

![符号変化あり→極値成立（左）と符号変化なし→極値なし・反例 x³（右）](/assets/images/diff-increasing-decreasing-combined.png)

左パネル: \\( f'(x) \\) が正→負に変化する点で極大、負→正で極小が成立する。右パネル: \\( f'(0) = 0 \\) でも \\( x^3 \\) はずっと増加しており、\\( x = 0 \\) は極値でない。

---

## なぜ \\( f'(c) = 0 \\) は必要条件にすぎないか

**\\( f'(c) = 0 \\) は極値であるための必要条件**ですが十分条件ではありません。\\( f'(c) = 0 \\) だとしても、\\( c \\) の前後で \\( f'(x) \\) の符号が変わらなければ、関数はその点を通過するだけで極値を持ちません。

**反例: \\( f(x) = x^3 \\)**

\\( f'(x) = 3x^2 \\) なので \\( f'(0) = 0 \\) です。しかし \\( x \neq 0 \\) では常に \\( f'(x) = 3x^2 > 0 \\) なので、\\( f(x) \\) は \\( x = 0 \\) の前後でずっと増加しています。\\( x = 0 \\) は極値でありません。

**極値の判定: 符号変化の確認が必須**

- \\( f'(x) \\) の符号が \\( + \to - \\) → **極大**
- \\( f'(x) \\) の符号が \\( - \to + \\) → **極小**
- \\( f'(c) = 0 \\) でも符号変化なし → **極値でない**

---

## 場合別の計算

### (1) \\( f(x) = -x^3 + 3x \\)

**Step 1:** 導関数を求める。

<div>
$$
f'(x) = -3x^2 + 3 = -3(x^2 - 1) = -3(x+1)(x-1)
$$
</div>

**Step 2:** \\( f'(x) = 0 \\) を解く。

<div>
$$
x = -1,\quad x = 1
$$
</div>

**Step 3:** 増減表を書く。

| \\( x \\) | \\( \cdots \\) | \\( -1 \\) | \\( \cdots \\) | \\( 1 \\) | \\( \cdots \\) |
|---|---|---|---|---|---|
| \\( f'(x) \\) | \\( - \\) | \\( 0 \\) | \\( + \\) | \\( 0 \\) | \\( - \\) |
| \\( f(x) \\) | ↘ | 極小 | ↗ | 極大 | ↘ |

**Step 4:** 極値を計算する。

<div>
$$
f(-1) = -(-1)^3 + 3(-1) = 1 - 3 = -2 \quad \text{（極小値）}
$$
</div>

<div>
$$
f(1) = -(1)^3 + 3(1) = -1 + 3 = 2 \quad \text{（極大値）}
$$
</div>

### (2) \\( f(x) = x^3 \\)

**Step 1:** 導関数を求める。

<div>
$$
f'(x) = 3x^2
$$
</div>

**Step 2:** \\( f'(x) = 0 \\) を解く。

<div>
$$
x = 0
$$
</div>

**Step 3:** 増減表を書く。

| \\( x \\) | \\( \cdots \\) | \\( 0 \\) | \\( \cdots \\) |
|---|---|---|---|
| \\( f'(x) \\) | \\( + \\) | \\( 0 \\) | \\( + \\) |
| \\( f(x) \\) | ↗ | ↗ | ↗ |

\\( x = 0 \\) の前後で \\( f'(x) \\) の符号は \\( + \\) のまま変化しないので、**極値なし**。

---

## まとめ

| 手順 | 内容 |
|---|---|
| 1. 導関数を求める | \\( f'(x) = \cdots \\) |
| 2. \\( f'(x) = 0 \\) の解を求める | 候補点 \\( x = c \\) |
| 3. 符号変化を確認する | 前後の \\( f'(x) \\) の正負を調べる |
| 4. 極値を計算する | \\( f(c) \\) が極大値または極小値 |

増減表を書くとき、\\( f'(c) = 0 \\) だからといって極値と決めつけないこと。\\( f'(x) \\) の符号が変わるかどうかを必ず確認するのが正しい手順です。
