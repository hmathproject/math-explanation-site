# 指数関数の平行移動と対称移動

---

## 元のグラフを確認する

\\( a > 0,\ a \neq 1 \\) のとき、\\( y = a^x \\) のグラフは次の性質を持ちます。

- 点 \\( (0,\ 1) \\) を通る（\\( a^0 = 1 \\)）
- 漸近線 \\( y = 0 \\)（\\( y > 0 \\) で x 軸に交わらない）
- \\( a > 1 \\) なら単調増加、\\( 0 < a < 1 \\) なら単調減少

グラフの詳細は[指数関数のグラフ](/exp-function-graph/)で確認できます。このページでは、このグラフを動かす操作（平行移動・対称移動）を見ていきます。

---

## 平行移動

\\( y = a^x \\) のグラフを平行移動すると次の式になります。

**x 軸方向に \\( p \\) だけ移動**

<div>
$$
y = a^{x-p}
$$
</div>

\\( p > 0 \\) なら右、\\( p < 0 \\) なら左に移動します。

例：\\( y = 2^{x-1} \\) は \\( y = 2^x \\) を**右に 1** だけ移動したグラフ。

**y 軸方向に \\( q \\) だけ移動**

<div>
$$
y = a^x + q
$$
</div>

\\( q > 0 \\) なら上、\\( q < 0 \\) なら下に移動します。y 方向に移動すると、漸近線も \\( y = 0 \\) から \\( y = q \\) に変わります。

例：\\( y = 2^x + 1 \\) は \\( y = 2^x \\) を**上に 1** だけ移動したグラフ（漸近線は \\( y = 1 \\)）。

**両方向への移動**

<div>
$$
y = a^{x-p} + q
$$
</div>

x 方向に \\( p \\)、y 方向に \\( q \\) だけ移動したグラフ。漸近線は \\( y = q \\)。

---

## 対称移動

**y 軸対称**

\\( y = a^x \\) を y 軸に関して対称移動するには、\\( x \\) を \\( -x \\) に置き換えます。

<div>
$$
y = a^{-x}
$$
</div>

この式には重要な見方があります。

<div>
$$
a^{-x} = \left(\frac{1}{a}\right)^x
$$
</div>

つまり **\\( y = a^{-x} \\) は、底を逆数にした \\( y = \left(\dfrac{1}{a}\right)^x \\) と同じグラフ**です。

例：\\( y = 2^{-x} = \left(\dfrac{1}{2}\right)^x \\)

\\( a > 1 \\) の底を持つ指数関数 \\( y = a^x \\) の y 軸対称は、\\( 0 < \dfrac{1}{a} < 1 \\) の底を持つ右下がりのグラフになります。増減が入れ替わるのは、底の逆数をとることで \\( a > 1 \\) と \\( 0 < a < 1 \\) が入れ替わるためです。

![指数関数の平行移動と対称移動](/assets/images/exp-function-transform.png)

左のグラフ：平行移動の例。\\( y = 2^{x-1} \\) は右に 1、\\( y = 2^x + 1 \\) は上に 1 移動したグラフです（漸近線が \\( y = 1 \\) に変わっています）。

右のグラフ：y 軸対称。\\( y = 2^{-x} \\) と \\( y = \left(\dfrac{1}{2}\right)^x \\) は同じ曲線で、\\( y = 2^x \\) と y 軸について対称な形になっています。

---

**補足：x 軸対称・原点対称**

- x 軸対称：\\( y = -a^x \\)（常に \\( y < 0 \\)、漸近線 \\( y = 0 \\) の下側）
- 原点対称：\\( y = -a^{-x} = -\left(\dfrac{1}{a}\right)^x \\)
