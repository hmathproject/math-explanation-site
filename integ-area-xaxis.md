# x軸との面積 — 符号と絶対値、区間分割の必要性

---

## 定積分と面積の違い

定積分 \\( \int_a^b f(x)\,dx \\) は \\( f(x) \\) が \\( x \\) 軸の下（\\( f(x) < 0 \\)）にある区間では**負の値**を返します。「面積」は常に 0 以上の量ですから、\\( f(x) \\) が \\( x \\) 軸をまたぐ場合、単純に定積分を計算しても面積にはなりません。

**具体例で確認:** \\( \int_{-1}^{2}(x^2 - 1)\,dx \\) を計算すると

<div>
$$
\int_{-1}^{2}(x^2-1)\,dx = \left[\frac{x^3}{3} - x\right]_{-1}^{2} = \left(\frac{8}{3}-2\right) - \left(-\frac{1}{3}+1\right) = \frac{2}{3} - \frac{2}{3} = 0
$$
</div>

しかしグラフを見ると、\\( y = x^2 - 1 \\) は \\( x \\) 軸と \\( x = \pm 1 \\) で交わり、\\( [-1,\,1] \\) では \\( x \\) 軸の下にあります。面積がゼロのはずはなく、この \\( 0 \\) は「正の面積と負の面積が打ち消し合った結果」です。

---

## 面積の求め方 — 4 ステップ

**Step 1:** \\( f(x) = 0 \\) を解いて \\( x \\) 軸との交点を求める。

**Step 2:** 各区間で \\( f(x) \\) の符号を確認する。

**Step 3:** 各区間の定積分を計算し、\\( f(x) < 0 \\) の区間では絶対値を取る（符号を反転）。

**Step 4:** 各区間の結果を合算する。

---

## 計算例: \\( y = x^2 - 1 \\) と \\( x \\) 軸の間の面積（\\( -1 \leq x \leq 2 \\)）

**Step 1:** 交点を求める。

<div>
$$
x^2 - 1 = 0 \implies x = \pm 1
$$
</div>

**Step 2:** 符号の確認。

- \\( -1 \leq x \leq 1 \\) のとき: \\( x^2 - 1 \leq 0 \\)（\\( x \\) 軸の下）
- \\( 1 \leq x \leq 2 \\) のとき: \\( x^2 - 1 \geq 0 \\)（\\( x \\) 軸の上）

**Step 3:** 各区間を計算する。

\\( [-1,\,1] \\) での定積分:

<div>
$$
\int_{-1}^{1}(x^2-1)\,dx = \left[\frac{x^3}{3}-x\right]_{-1}^{1} = \left(\frac{1}{3}-1\right)-\left(-\frac{1}{3}+1\right) = -\frac{2}{3} - \frac{2}{3} = -\frac{4}{3}
$$
</div>

\\( f < 0 \\) なので絶対値: \\( \left|-\frac{4}{3}\right| = \frac{4}{3} \\)

\\( [1,\,2] \\) での定積分:

<div>
$$
\int_{1}^{2}(x^2-1)\,dx = \left[\frac{x^3}{3}-x\right]_{1}^{2} = \left(\frac{8}{3}-2\right)-\left(\frac{1}{3}-1\right) = \frac{2}{3}-\left(-\frac{2}{3}\right) = \frac{4}{3}
$$
</div>

\\( f \geq 0 \\) なのでそのまま: \\( \frac{4}{3} \\)

**Step 4:** 合算する。

<div>
$$
S = \frac{4}{3} + \frac{4}{3} = \frac{8}{3}
$$
</div>

**答え:** \\( S = \frac{8}{3} \\)

区間分割なしで計算した \\( 0 \\) とは大きく異なります。これが区間分割が必須である理由です。

---

## 絶対値記法による一般式

<div>
$$
S = \int_a^b |f(x)|\,dx
$$
</div>

これが面積の正確な定義です。\\( f(x) \\) の符号が途中で変わらない区間では絶対値を外して直接積分でき、符号が変わる点（\\( x \\) 軸との交点）で区間を分割してから各区間を計算・合算します。
