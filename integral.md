# 積分

積分は微分の逆演算として定義されます。\\( f'(x) \\) から \\( f(x) \\) を「戻す」操作が不定積分であり、その結果は一つの関数ではなく積分定数 \\( C \\) を含む関数の族になります。

積分の真価が発揮されるのは**定積分**においてです。定積分 \\( \int_a^b f(x)\,dx \\) は、微積分の基本定理によって「原始関数の差 \\( F(b) - F(a) \\)」として計算できます。そしてこの値は、\\( y = f(x) \\) と \\( x \\) 軸の間の「符号付き面積」に等しいという幾何学的な意味を持ちます。符号付き面積という視点を押さえることで、面積の問題における区間分割の必然性と 1/6 公式の成立理由が自然に理解できます。
