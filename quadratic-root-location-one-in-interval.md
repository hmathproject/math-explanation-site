# 解の配置（解が区間を挟む）— \\( f(\\alpha) \\cdot f(\\beta) < 0 \\) の1条件で決まる

---

## 問題

方程式 \\( x^2 - 4x + a = 0 \\) が、区間 \\( (0,\ 3) \\) の内側と外側に1解ずつを持つ実数 \\( a \\) の範囲を求めよ。

---

## 条件の読み取り

「解が区間 \\( (0, 3) \\) を挟む」とは、2解の一方が \\( 0 < x < 3 \\) の内側、もう一方が \\( 0 < x < 3 \\) の外側にある状態です。

放物線で考えると、\\( x = 0 \\) と \\( x = 3 \\) の間でx軸を横切り、さらにその外でもう1回横切ります。つまり放物線は \\( x = 0 \\) と \\( x = 3 \\) の「片方でx軸より上、もう片方でx軸より下」になっています。

---

## 解が区間を挟む1条件

\\( f(0) \\) と \\( f(3) \\) の符号が逆——すなわち \\( f(0) \cdot f(3) < 0 \\) が成り立てば、放物線は \\( x = 0 \\) と \\( x = 3 \\) の間で必ずx軸を横切ります。これが解が区間内に1つ存在することを保証します。

連続な関数が \\( x = 0 \\) と \\( x = 3 \\) で異なる符号をとるならば、中間値の定理によりその間で必ず0になる点が存在します——これが「区間内の解」です。

下の図で、\\( f(0) > 0 \\) かつ \\( f(3) < 0 \\)（左）と \\( f(0) < 0 \\) かつ \\( f(3) > 0 \\)（右）の2パターンを確認してください。どちらの場合も区間 \\( (0, 3) \\) の内と外に1解ずつあることが見えます。

![解の配置（解が区間を挟む）：f(0)とf(3)が異符号の2パターン](/assets/images/quadratic-root-location-one-in-interval-combined.png)

---

## なぜ判別式・軸の条件が不要か

前の記事（2解がともに区間内）では判別式・軸・端点の値という3条件が必要でした。しかし今回は \\( f(0) \cdot f(3) < 0 \\) の1条件だけで十分です。

\\( f(0) \cdot f(3) < 0 \\) が成り立つ時点で、2解の存在（実数解が2つある）は自動的に保証されています。また解の片方が \\( (0, 3) \\) の外側にある以上、軸がどこにあっても「1解が内側・1解が外側」という配置は確定します。

---

## 前の記事との比較

| 条件 | 2解がともに区間内 | 解が区間を挟む |
|---|---|---|
| 判別式 \\( D \geq 0 \\) | 必要 | 不要（自動的に保証） |
| \\( f(\alpha) > 0 \\) かつ \\( f(\beta) > 0 \\) | 必要 | 代わりに \\( f(\alpha) \cdot f(\beta) < 0 \\) |
| \\( \alpha < \\) 軸 \\( < \beta \\) | 必要 | 不要 |

---

## まとめ

<div>
$$
\text{解が区間 } (\alpha, \beta) \text{ を挟む}
\iff f(\alpha) \cdot f(\beta) < 0
$$
</div>

この問題では \\( f(0) = a \\)、\\( f(3) = a - 3 \\) なので：

<div>
$$
f(0) \cdot f(3) < 0 \iff a(a-3) < 0 \iff 0 < a < 3
$$
</div>

1条件の単純な計算で答えが出る構造は、「解が区間を挟む」ならではの特徴です。前の記事（2解がともに区間内、答え \\( 3 < a \leq 4 \\)）と見比べると、同じ放物線でも条件が変わると答えと計算量がどう変わるかが確認できます。
