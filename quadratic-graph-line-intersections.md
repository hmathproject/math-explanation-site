# 放物線と直線の共有点の個数 — 判別式 \\( D \\) の符号で決まる

---

## 問題

放物線 \\( y = x^2 \\) と直線 \\( y = x + k \\)（\\( k \\) は定数）の共有点の個数を、\\( k \\) の値によって場合分けせよ。

---

## なぜ判別式で個数がわかるか

放物線と直線の交点の \\( x \\) 座標は、2つの式を連立した方程式の解です。

<div>
$$
x^2 = x + k \implies x^2 - x - k = 0
$$
</div>

この2次方程式 \\( x^2 - x - k = 0 \\) の実数解の個数が、そのまま共有点の個数になります。

判別式 \\( D = (-1)^2 - 4 \cdot 1 \cdot (-k) = 1 + 4k \\) の符号によって：

| \\( D \\) の符号 | 実数解の個数 | 共有点の個数 |
|---|---|---|
| \\( D > 0 \\) | 2つ | 2点 |
| \\( D = 0 \\) | 1つ（重解） | 1点（接する） |
| \\( D < 0 \\) | 実数解なし | 0点 |

下の図でそれぞれの場合を確認してください。

![放物線と直線の共有点の個数（D>0/D=0/D<0）](/assets/images/quadratic-graph-line-intersections-combined.png)

---

## 場合分けの解答

判別式 \\( D = 1 + 4k \\) の符号を \\( k \\) で表します。

<div>
$$
D = 1 + 4k
\begin{cases}
> 0 & \left(k > -\dfrac{1}{4}\right) \\[8pt]
= 0 & \left(k = -\dfrac{1}{4}\right) \\[8pt]
< 0 & \left(k < -\dfrac{1}{4}\right)
\end{cases}
$$
</div>

したがって：

- \\( k > -\frac{1}{4} \\) のとき：**共有点は2個**
- \\( k = -\frac{1}{4} \\) のとき：**共有点は1個**（直線は放物線に接する）
- \\( k < -\frac{1}{4} \\) のとき：**共有点は0個**

**接するとき（D = 0）の接点の座標**：

\\( k = -\frac{1}{4} \\) のとき、\\( x^2 - x + \frac{1}{4} = 0 \\) を解くと \\( \left(x - \frac{1}{2}\right)^2 = 0 \\) なので \\( x = \frac{1}{2} \\)、\\( y = \frac{1}{4} \\)。接点は \\( \left(\frac{1}{2},\ \frac{1}{4}\right) \\)。

---

## まとめ

**共有点の個数を求める手順**：

1. 2式を連立して \\( ax^2 + bx + c = 0 \\) の形にまとめる
2. 判別式 \\( D = b^2 - 4ac \\) を計算する
3. \\( D > 0 \\)（2個）、\\( D = 0 \\)（1個・接する）、\\( D < 0 \\)（0個）で場合分けする

| 条件 | 共有点の個数 |
|---|---|
| \\( k > -\frac{1}{4} \\)（\\( D > 0 \\)） | 2個 |
| \\( k = -\frac{1}{4} \\)（\\( D = 0 \\)） | 1個（接する） |
| \\( k < -\frac{1}{4} \\)（\\( D < 0 \\)） | 0個 |
