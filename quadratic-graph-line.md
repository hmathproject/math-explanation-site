# 二次関数のグラフと直線

**放物線と直線の交わり方を、判別式・連立方程式・積分の3ステップで理解する**

---

## この単元で学ぶこと

放物線 \\( y = ax^2 + bx + c \\) と直線 \\( y = mx + n \\) の交点に関する問題には、大きく3つのタイプがあります。

- **共有点の個数**：交わるかどうか、何点で交わるかを調べる。判別式が道具になります。
- **共有点の座標**：どこで交わるかを求める。連立方程式を立てて解きます。
- **囲まれた面積**：放物線と直線に囲まれた図形の面積を求める。積分（定積分）を使います。

この3つは密接につながっています。「交わる点がある（D > 0）」ことを確認してから「座標を求め」、必要なら「面積を計算する」という流れが基本です。

## 解説記事

### [放物線と直線の共有点の個数](/quadratic-graph-line-intersections/)

放物線 \\( y = x^2 \\) と直線 \\( y = x + k \\) の共有点の個数を、\\( k \\) の値によって場合分けします。

「交わる点が何個あるか」は、連立して得られる2次方程式の判別式 \\( D \\) の符号で決まります。\\( D > 0 \\) なら2点、\\( D = 0 \\) なら1点（接する）、\\( D < 0 \\) なら0点です。

---

### [放物線と直線の共有点の座標](/quadratic-graph-line-coordinates/)

放物線 \\( y = x^2 \\) と直線 \\( y = 2x + 3 \\) の共有点の座標を求めます。

連立すると \\( x^2 = 2x + 3 \\) となり、因数分解で \\( x = -1, 3 \\) が出ます。それぞれの \\( y \\) 値を求めることで共有点の座標が確定します。

---

### [放物線と直線で囲まれた面積](/quadratic-graph-line-area/)

放物線 \\( y = x^2 \\) と直線 \\( y = x + 2 \\) で囲まれた図形の面積を求めます。

まず共有点（積分の端点）を求め、「直線 \\( - \\) 放物線」を積分します。公式 \\( S = \dfrac{|a|}{6}(\\beta - \\alpha)^3 \\) を使う方法も合わせて確認します。
