# 図形を含む二次関数の問題 — 条件を方程式に変換して解く

---

## 共通の方針

図形の問題では、「面積・周長・距離などの条件が等式として与えられる」ことが多く、その等式が二次方程式になります。

手順：
1. **変数を決める**：求めたい量（または一方の辺の長さ）を \\( x \\) とおく
2. **他の量を \\( x \\) で表す**：条件をすべて \\( x \\) の式にする
3. **方程式を立てる**：等式条件（面積 = ○、周長 = ○ など）を式にする
4. **解いて吟味する**：得られた解が文脈上ありえる値かを確認する

---

## 問題1：道幅問題

### 問題

縦30m、横40mの長方形の土地に、幅 \\( x \\) m の道路を縦横に1本ずつ作ったところ、道路を除いた面積が \\( 875 \\) m² になった。道幅 \\( x \\) を求めよ。

### 解き方

下の図（左）は縦横に道が通った状態の模式図です。

![道幅問題の模式図と長方形問題のグラフ](/assets/images/quadratic-word-problems-geometry-combined.png)

縦道（幅 \\( x \\) m）と横道（幅 \\( x \\) m）を取り除いた残りの面積は：

<div>
$$
S(x) = (30 - x)(40 - x)
$$
</div>

これが \\( 875 \\) m² になるので：

<div>
$$
(30 - x)(40 - x) = 875
$$
</div>

展開します：

<div>
$$
x^2 - 70x + 1200 = 875
$$
</div>

<div>
$$
x^2 - 70x + 325 = 0
$$
</div>

因数分解します：

<div>
$$
(x - 5)(x - 65) = 0
$$
</div>

\\( x = 5 \\) または \\( x = 65 \\)。

**吟味**：道幅は土地の縦（30m）より小さいので \\( 0 < x < 30 \\)。したがって \\( x = 5 \\)。

**答え**：道幅は **5m**。

**確認**：\\( (30-5)(40-5) = 25 \times 35 = 875 \\) ✓

## 問題2：長方形問題

### 問題

縦の長さが横の長さより \\( 4 \\) cm 長い長方形がある。面積が \\( 45 \\) cm² のとき、縦と横の長さをそれぞれ求めよ。

### 解き方

横の長さを \\( x \\) cm（\\( x > 0 \\)）とおく。縦の長さは \\( (x + 4) \\) cm。

面積が45 cm²なので：

<div>
$$
x(x + 4) = 45
$$
</div>

展開します：

<div>
$$
x^2 + 4x - 45 = 0
$$
</div>

因数分解します：

<div>
$$
(x + 9)(x - 5) = 0
$$
</div>

\\( x = -9 \\) または \\( x = 5 \\)。

**吟味**：\\( x > 0 \\) なので \\( x = 5 \\)。

縦は \\( 5 + 4 = 9 \\) cm。

**答え**：横 **5cm**、縦 **9cm**。

**確認**：\\( 5 \times 9 = 45 \\) ✓、縦と横の差 \\( = 9 - 5 = 4 \\) ✓

## まとめ：図形問題を解く定石

| ステップ | 内容 |
|---|---|
| ① 変数設定 | 求めたい辺・幅などを \\( x \\) とおく（正の整数・正の実数の範囲を明記） |
| ② 他の量を式化 | 関係式から他の辺・量を \\( x \\) で表す |
| ③ 方程式を立てる | 面積・周長などの等式条件を \\( x \\) の方程式にする |
| ④ 方程式を解く | 因数分解・解の公式などで解く |
| ⑤ 解を吟味する | \\( x > 0 \\)・\\( x < \text{限界値} \\) など現実的な条件で絞る |
